# Security Policy

## Status

DesktopRanger is an experimental research prototype. It is provided for defensive security research, Windows isolation experiments, and engineering discussion.

The project is not production-ready and does not provide commercial security support or guaranteed response timelines.

## Reporting

If you want to report a security issue related to this repository, please avoid disclosing sensitive technical details in public GitHub issues.

For private contact, use:

https://igorkorkin.github.io/

## Scope

This public repository contains a limited defensive prototype.

Internal evaluation materials, private test harnesses, experimental logs, and unpublished research artifacts are maintained separately and are outside the scope of this public repository.

## Public Issues

Public GitHub issues may be used for:

* build problems;
* documentation errors;
* reproducibility questions;
* defensive engineering discussion;
* non-sensitive bug reports.
